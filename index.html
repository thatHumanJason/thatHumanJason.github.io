Python Turtle Clock
Description

A large flexible clock object written in python 3 and using the turtle class, this object creates a live clock with several preset styles with the ability to create your own custom style for a unique analog clock face.


Purpose

This repository was created so that I could practice creating large, complex objects that are very customizable.
